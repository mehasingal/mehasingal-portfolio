MoodRoom

MoodRoom is a cozy 3D room decoration game and DBT-aligned habit tracker designed to help young adult patients engage with behavioral and emotional regulation skills set by their therapists — making clinical goal-tracking feel less like a chore and more like play. As Lead Engineer and the sole developer on a cross-functional team of five, I joined the project after development had stalled and immediately got to work triaging bugs across multiple gameplay systems to revive the MVP. I translated Figma designs into fully functional in-engine UI and gameplay features using GDScript in the Godot engine, and redesigned the modular game architecture from the ground up to support a growing library of 3D assets and media. Within one month, I shipped a fully functional prototype to early testers via Apple TestFlight for clinical feedback.

TerraAntiqua

TerraAntiqua grew out of a frustration with history textbooks that felt strikingly Eurocentric, where civilizations were taught as isolated stories rather than chapters in a shared global timeline. The app reimagines history as a living, interconnected map you can explore, letting users travel through ancient civilizations anchored to real geography and a scrollable timeline. Built with React and the Google Maps API, TerraAntiqua was developed as a fully AI-native app using Claude Code, accelerating development across the full stack. One of the most meaningful parts of the project was manually reconstructing fragmented historical data into clean GeoJSON datasets — work I've contributed back to the open source community for other developers and mapmakers to use.

  • React
  • Google Maps API
  • GeoJSON
  • Claude Code

Snohomish County Disaster Relief Platform

Built for a senior capstone course with a real client — the Snohomish County Department of Emergency Management — this FEMA-compliant Survey123 application integrated with ArcGIS Enterprise and Tableau dashboards gives emergency staff real-time visibility into disaster impact across a region of 800,000+ residents. As technical lead of a three-person team, I built the Excel automation and logic powering the survey's FEMA compliance structure. UW Alert Mapping System

The UW Alert Mapping System started as a class project and evolved into a working prototype pitched to the UW Alert Director as a potential tool for improving campus safety communication for students. The platform processes live 911 call data from open-source APIs, visualizing incidents as map pins with surrounding buffer zones that flag high-risk areas, and providing turn-by-turn safe route navigation around them. As technical and full-stack lead of a five-person team, I handled all API integration and backend data pipelines, with the geospatial analysis powered by Turf.js and Mapbox on the frontend and Python and pandas on the backend. The platform is currently live as a desktop web app, with an iOS Swift extension in active development.

  • Mapbox
  • Turf.js

Parsing the English Catalogue

research-1.jpg
research-2.jpg
research-3.jpg

As one of 16 undergraduate research fellows at the UW eScience Institute, I built automated Python text-parsing pipelines using regular expressions to process a 160,000-record bibliographic dataset of early 20th-century British publishing records digitized by HathiTrust. A core part of the work involved inspecting and tinkering with the underlying OCR machine learning algorithms — identifying where the model consistently failed and adjusting extraction logic to compensate — improving data coverage by 15% and achieving 98% extraction accuracy. I also designed a normalized SQL schema to store and query the processed records, increasing analysis speed by 20%.
